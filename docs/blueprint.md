# **App Name**: Harish Mohanani - Portfolio

## Core Features:

- Hero Section: Displays name, title, years of experience, a short intro, and prominent 'View Projects' and 'Download Resume' CTAs.
- About Section: Presents a concise description highlighting product thinking, UI architecture, performance & scalability expertise, and an ownership mindset.
- Projects Showcase: Displays 4-6 project cards, each featuring a project title, concise problem statement, tech stack, key technical challenges, and links to a Live Demo and the GitHub repository.
- Skills Summary: Categorizes skills into Frontend, Backend, Tooling, and Testing sections, presenting a concise and readable summary.
- Experience Timeline: Provides a high-level timeline emphasizing impact, ownership, and leadership roles at companies like Exabeam, GlobalLogic, AppDirect, Rocket Software, and Coditas.
- Contact Information: Presents contact information, including email, GitHub, and LinkedIn profiles.

## Style Guidelines:

- Primary color: Indigo (#4F46E5) for a professional, trustworthy, and modern look.
- Background color: Very light gray (#F9FAFB), providing a clean, light aesthetic with excellent contrast.
- Accent color: Sky blue (#38bdf8), an analogous color to Indigo that highlights CTAs and interactive elements, adding a subtle, modern touch.
- Body and headline font: 'Inter', a sans-serif typeface with a neutral, readable appearance, perfect for both headlines and body text.
- Use a consistent set of simple, line-based icons for skills and contact information, to maintain a minimal and clean aesthetic.
- Employs a one-page layout with generous spacing, ensuring readability and a clean, editorial feel. Sections are clearly separated, maintaining a structured flow.
- Incorporate subtle fade-in effects on scroll and hover effects on project cards and buttons for a polished, non-intrusive user experience.
